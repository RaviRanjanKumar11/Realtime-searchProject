#  Real-Time Search System (Laravel + ElasticSearch)

A production-style real-time search application built with **Laravel**, **MySQL**, **ElasticSearch**, **Redis**, and **Bootstrap**.  
Designed to demonstrate scalable search architecture for large datasets (1M+ records).

---

##  Features

-  Authentication (Laravel Breeze)
-  Login with Google OAuth
-  MySQL database with **1,000,000+ seeded records**
-  Real-time search using **ElasticSearch**
-  Redis caching for search results
-  Admin email notification on every search keyword
-  Fault-tolerant design (ElasticSearch â†’ MySQL fallback)
-  Simple UI using HTML, Bootstrap & JavaScript
